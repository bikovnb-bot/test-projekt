📂 Импорт показаний из Excel

Инструкция

Требования к файлу Excel (.xlsx):

  • Первая строка – заголовки колонок.
  • Обязательные колонки: meter_serial (серийный номер прибора учёта), date (дата в формате YYYY-MM-DD).
  • Для однотарифных счётчиков нужна колонка value (показание).
  • Для многотарифных счётчиков нужно добавить колонки с названиями зон (например, «День», «Ночь»), они должны точно совпадать с названиями компонентов тарифа в системе.
  • Если счётчик многотарифный, колонка value игнорируется.
  • Если показание за указанную дату уже существует, оно будет обновлено.

Пример заполнения:

  • для однотарифного: meter_serial, date, value
  • для многотарифного: meter_serial, date, День, Ночь
