📖 Роли пользователей и их права

Каждая роль определяет набор действий в различных приложениях системы.

{% for role in roles_info %}

{{ role.name }}

Права по приложениям:
🏢 Договоры (эксплуатация)
    {% if role.code == 'ADMIN' %}
  • Полный CRUD договоров, оплат, дополнительных соглашений
  • Импорт/экспорт Excel, управление шаблонами
  • Просмотр любых договоров
    • {% elif role.code == 'MANAGER' %}
  • Создание, редактирование договоров (кроме удаления)
  • Добавление и редактирование оплат, загрузка документов
  • Просмотр всех договоров
    • {% elif role.code == 'VIEWER' %}
  • Только просмотр договоров, оплат, отчётов
  • Экспорт в Excel (только чтение)
    • {% elif role.code == 'CONTRACTOR' %}
  • Доступ только к своим договорам (по полю «исполнитель»)
  • Может добавлять оплаты и документы к своим договорам
  • Просмотр только своих отчётов
    • {% endif %}
⚡ Энергоучёт
    {% if role.code == 'ADMIN' or role.code == 'MANAGER' %}
  • Управление приборами учёта (CRUD)
  • Импорт показаний, аномалии, графики
  • Просмотр любых показаний
    • {% elif role.code == 'VIEWER' %}
  • Просмотр приборов и показаний, отчётов
    • {% elif role.code == 'CONTRACTOR' %}
  • Просмотр только своих приборов (если привязаны)
    • {% endif %}
📋 Заявки
    {% if role.code == 'ADMIN' or role.code == 'MANAGER' %}
  • Управление заявками (создание, назначение, закрытие)
  • Управление справочниками (типы заявок, материалы)
  • Просмотр всех заявок
    • {% elif role.code == 'VIEWER' %}
  • Просмотр всех заявок без возможности изменения
    • {% elif role.code == 'CONTRACTOR' %}
  • Может создавать заявки только для своих договоров
  • Просмотр только своих заявок
    • {% endif %}
👥 Пользователи
    {% if role.code == 'ADMIN' %}
  • Управление пользователями, ролями, группами
  • Смена паролей, активация/деактивация
    • {% elif role.code == 'MANAGER' %}
  • Просмотр списка пользователей (без редактирования)
    • {% else %}
  • Нет доступа
    • {% endif %}
{% endfor %}
Примечание: Группы могут расширять права (например, давать доступ к определённым отчётам). Назначение групп происходит через редактирование пользователя.
