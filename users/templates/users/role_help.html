🏢 Договоры

  • Создание, редактирование договоров (кроме удаления)

    {% if role.code == 'ADMIN' or role.code == 'CONTRACT_SPECIALIST' or role.code == 'ENGINEER' %}
  • Управление приборами учёта (CRUD)
  • Импорт показаний, аномалии, графики
    • {% else %}
  • Нет доступа
    • {% endif %}
📋 Заявки
    {% if role.code == 'ADMIN' or role.code == 'ENGINEER' or role.code == 'DISPATCHER' %}
  • Полный доступ к заявкам (создание, назначение, закрытие)
    • {% elif role.code == 'WORKER' %}
  • Доступ только к назначенным заявкам
    • {% else %}
  • Нет доступа
    • {% endif %}

    • {% else %}

    • {% endif %}
